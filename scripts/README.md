## Scripts Overview

- **`compare_tokenisers.ipynb`** – Computes subword fertility rates for each language and model.
- 
- **`create_folds.py`** – Generates random train/dev/test folds for multitask and multiclass setups.   

- **`downsample_train_sets.ipynb`** – Splits the training set into smaller sub-folds to experiment with the impact of different training sizes on model performance.  

- **`eval_near_misses.ipynb`** – Calculates precision (P), recall (R), and F1 scores, both in strict and lenient evaluation modes.
